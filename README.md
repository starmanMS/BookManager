## 图书管理系统

### 开发环境

- Spring + MySQL + Mybatis + SpringSecurity + Thymeleaf + Tomcat

### 项目功能

- 新增注册功能 权限划分 普通用户和管理员

- 管理员可以进行图书的添加与删除

- 用户可以进行图书的借阅和归还
